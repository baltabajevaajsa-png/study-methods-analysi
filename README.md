# study-methods-analysi
Analysis of how different study methods affect student performance

## 📌 Topic
Impact of study methods on academic performance

## 🎯 Goal
To analyze how different learning methods affect students' results.

## 📊 Data
Survey of students including:
- Study time
- Learning methods
- Academic performance

## 📈 Methods
- Data visualization
- Comparative analysis

## 🔍 Key Findings
- Active learning methods are more effective
- Passive learning leads to lower performance
